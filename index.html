Política de Privacidade — PromoLar Ofertas

Última atualização: 16 de março de 2026

1. Quem somos

O PromoLar Ofertas é um serviço automatizado que divulga ofertas de produtos de casa e cozinha no Instagram (@promo_lar_ofertas) e no Telegram (@PromoLarOfertas). Quando você comenta em uma publicação, podemos enviar o link da oferta por mensagem direta (DM).

2. Quais dados coletamos

  • ID de usuário do Instagram — para enviar a DM com o link da oferta.
  • ID do comentário — para responder ao comentário com uma confirmação.
  • ID da publicação — para identificar qual oferta você solicitou.

Não coletamos nome, e-mail, telefone nem nenhuma outra informação pessoal.

3. Como usamos os dados

Os dados são usados exclusivamente para:

  • Enviar uma única DM com o link afiliado da oferta que você comentou.
  • Evitar o envio duplicado da mesma mensagem.

Não compartilhamos, vendemos nem usamos seus dados para publicidade.

4. Por quanto tempo armazenamos

Os IDs de usuário e comentário são armazenados por até 7 dias após o envio da DM, apenas para evitar duplicatas. Após esse período são excluídos automaticamente.

5. Seus direitos

Você pode a qualquer momento:

  • Deixar de interagir com nossas publicações para não receber DMs.
  • Solicitar a exclusão dos seus dados enviando uma mensagem direta para @promo_lar_ofertas.

6. Contato

Dúvidas sobre esta política? Envie uma mensagem para o Instagram @promo_lar_ofertas.
